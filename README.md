# Tic Tac Toe Using React + TypeScript + Vite

Tic Tac Toe Using React + TypeScript + Vite, based on <https://react.dev/learn/tutorial-tic-tac-toe> + overachiever activities at the end of the article.

## Development

Start the server:
```shell
yarn server
```

Test the server:
```shell
yarn client
```

Start the app:
```shell
  yarn dev
```
